# 리포트아이 v15 — 1·2·3순위 통합 (FINAL)

## 🎯 이번 버전에서 해결된 것

### ⭐ 1순위 — 데이터 정확성
- **분석 호출 2단계 분리** (Phase 1: 데이터 / Phase 2: 서술)
- **44개 대학 권장과목 데이터** 통합 (대교협 + 연세·이화여대 공식)
- **이수 충족률 자동 계산** — 학생 grades와 학과별 권장과목 매칭
- **분량 강제 프롬프트** — 각 배열의 최소 개수, 각 필드 최소 글자수 명시

### ⭐ 2순위 — UI 강화
- **그래프 탭 5종** (전체 / 국영수 / 국영수사 / 국영수과 / 전공연계)
  - 희망 전공에 따라 자동 매핑 (이공계→수과, 인문→국사영 등)
  - 탭별 분석글 표시 (trendByGroup)
- **이수 현황 화면 완전 재작성** — 대학별 충족률 카드 + 분석 + 추천
- **강점·보완점 5개+ 모두 표시** (기존 slice 제거됨)
- **키워드 분석 강화** — 빈도 시각화 + 탐구방법 빈도 분류 + 분석글
- **학과 적합도** — actionPlan 추가, 5개+ 표시
- **탐구주제** — 학과별 매칭 + 기존 활동 연계 + 기대 결과
- **면접 질문** — 모범 답변·팁·주의 분리
- **종합 리포트 강화** — 부록 6개 추가 (역량·패턴·강점·학과·탐구·충족률)

### ⭐ 3순위 — 안정성
- **타임아웃 회피** — 클라이언트에서 Phase 1/2 별도 fetch
- **vercel.json** — 함수 maxDuration 60초 설정
- **합격 패턴 탭 제거** (s7) — 종합 리포트의 부록으로 통합

## ⚠️ 만약 에러 나면

1. **"Phase 1 서버 오류"** → Vercel 빌드 로그 확인 (Functions → Logs)
2. **타임아웃** → vercel.json 적용됐는지 확인 (Settings → Functions → maxDuration 60s 표시)
3. **데이터 비어있음** → 새로 분석 돌렸는지 확인 (이전 결과 캐시 가능)
4. **그래프 안 나옴** → Chart.js 로딩 확인 (index.html에 cdn 있는지)

문제 있으면 에러 메시지 + 스크린샷 보내주세요!
